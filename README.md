# Image URL Viewer - 图片链接提取工具

从文本中提取图片链接并可视化展示的工具。

## 功能

- 自动提取文本中的图片 URL
- 按行分组展示图片，方便对比
- 支持 Key-Value 标识（如 `sourceUrl:`, `targetUrl:`）
- 点击图片查看大图
- 一键复制单个或全部 URL
- 自定义布局（行数、列数）

## 使用

1. 在输入框粘贴包含图片 URL 的文本
2. 点击"提取并展示"按钮（或 `Ctrl/Cmd + Enter`）
3. 查看分组展示的图片

## 示例

```
sourceUrl: https://example.com/before.jpg
targetUrl: https://example.com/after.jpg
```

## 运行

直接用浏览器打开 `index.html` 即可使用。
